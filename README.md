# Node + Express Backend (From Scratch)

This project demonstrates how to build a simple backend server using Node.js and Express. It covers creating routes, handling JSON data, connecting to a frontend, and understanding middleware.

---

## 🚀 Features

- Setup Express server from scratch
- Create GET and POST routes
- Handle JSON data
- Use route parameters
- Connect backend to frontend
- Fix CORS issues
- Use Express Router for modular routing
- Understand and apply middleware

---

## 📚 Prerequisites

Before starting, you should be familiar with:

- JavaScript
- Async programming (Promises / async-await)
- Basic HTML

---

## ⚙️ Setup

1. Initialize project:

```bash
npm init -y
```

2. Install dependencies

```bash
npm install express cors
```

3. Run the server

```bash
npm start
```

4. You can use the VS Code Live Server extension to run your frontend locally.
